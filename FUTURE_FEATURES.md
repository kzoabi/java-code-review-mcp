# Future Features Wishlist

This document outlines planned future capabilities for the Java Code Review MCP Server.

---

## Phase 3: AI Integration

### 3.1 Code Generation
- Generate unit tests from code
- Generate factory methods
- Generate value object builders
- Generate record definitions from DTOs

### 3.2 Refactoring Suggestions
- Suggest refactoring based on JBCT violations
- Convert exceptions to Result/Promise
- Convert to records
- Extract method from complex lambdas

### 3.3 LLM-Powered Analysis
- Natural language code explanations
- Semantic security analysis
- Business logic review
- Code complexity scoring

---

## Phase 4: IDE/Tooling Integration

### 4.1 VS Code Extension
- Real-time JBCT hints
- Quick fix actions
- Inline annotations
- Code lens for JBCT issues

### 4.2 IntelliJ Plugin
- JBCT inspections
- Code style enforcement
- Live templates for JBCT patterns
- Intent actions for auto-fix

### 4.3 CI/CD Enhancements
- GitHub Action with auto-fix
- GitLab CI template
- Pre-commit hooks
- Jenkins pipeline integration

---

## Phase 5: Enhanced Reporting

### 5.1 Dashboard
- Trend analysis over time
- Team-wide statistics
- Issue classification by category
- Individual developer metrics

### 5.2 Export Formats
- HTML reports with charts
- PDF export
- Jira integration
- Linear integration

### 5.3 Notifications
- Slack integration
- Teams integration
- Email digests
- PR comments automation

---

## Priority Order

1. VS Code Extension (most requested)
2. GitHub Action with auto-fix
3. LLM-Powered Analysis
4. Dashboard
5. IntelliJ Plugin
